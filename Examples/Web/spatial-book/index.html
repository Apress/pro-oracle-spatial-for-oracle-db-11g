Simple Map

Using the JavaScript API (Oracle Maps)

  • ajax-map.html
    A simple free-scrolling map, without any decorations. Use your mouse to move around.
  • ajax-map-zoom.html
    Same with control panel added, allowing you to zoom in and out
  • ajax-map-decorated.html
    Overview map added, with some decorations: scalebar, logo, copyright notice
  • ajax-map-marquee.html
    Marquee zoom control: zoom by dragging a mouse rectangle
  • ajax-map-themes.html
    Show business information (customers, branches, competitors). Select the information to display. Click on any symbol to get details about the object.
  • ajax-map-searches.html
    Draw a circle, rectangle, or polygon on the map, then highlight those customers, branches or competitors that are inside the area..
